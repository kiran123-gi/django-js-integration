# Django JavaScript Integration Project

This is a simple project built using Django, HTML, and JavaScript to understand how frontend and backend interact.

## 🚀 Features
- Dynamic user interaction using JavaScript
- Integration with Django backend
- Improved user experience without full page reload

## 🧠 What I Learned
- How JavaScript works with Django templates
- Handling client-side events
- Connecting frontend actions with backend logic

## 🛠️ Tech Stack
- Python
- Django
- HTML
- JavaScript

## 📌 Project Goal
The main goal of this project is to understand how to integrate JavaScript with Django to build dynamic web applications.

## 🔗 Future Improvements
- Add AJAX for real-time data updates
- Improve UI design
- Add more interactive features
